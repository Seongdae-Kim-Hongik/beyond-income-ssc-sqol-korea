# Data

The microdata are **not** included in this repository because they are
distributed under the terms of the Korea Social Science Data Archive (KOSSDA).

To reproduce the analyses:

1. Obtain the **2021 Korean Happiness and Quality of Life Survey** (National
   Assembly Futures Institute) from KOSSDA, dataset **A1-2021-0003** (v2.0):
   https://doi.org/10.22687/KOSSDA-A1-2021-0003-V2.0
2. Save the data file in this folder as: `korean_happiness_qol_2021.csv`
   (the scripts read it with `EUC-KR` encoding).

No personally identifiable information is required or used.
